# StoryVerse Monorepo

This repository currently contains one app:

- `storyverse-web`: Next.js web client for StoryVerse

## Quick Start

```bash
cd storyverse-web
npm ci
npm run dev
```

Then open `http://localhost:3000`.

## Quality Gate

```bash
cd storyverse-web
npm run check
```

This runs lint, parser tests, and production build.
